第三方一些插件功能介绍

在项目中遇到一些问题可以使用某些插件来快速解决问题，记录一些插件的功能，以便后续在项目中使用（前端）

axios

安装

npm install axios

功能

  • 比Jquery轻量，但处理请求不多的时候，可以使用
  • 基于Promise语法标准
  • 支持nodejs
  • 自动转换JSON数据

    使用

    参考网站

classnames

安装

npm install classnames

功能

在使用jsx时，方便添加class

使用

参考网站

qs

安装

npm install qs

功能

在使用axios时，提交application/x-www-form-urlencoded格式的数据，qs.stringify(data)化后提交

使用

参考网站

rc-calendar

安装

npm install rc-calendar

功能

在使用rc-calendar，react提供的时间控件

使用

参考网站

cheerio

安装

npm install cheerio

功能

cheerio是在node把文档解析成dom，方便增删改查DOM，用于爬虫

使用

参考网站
